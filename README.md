# Курсовая работа по МО

Курсовая работа по классическому машинному обучению. 
Этот репозиторий содержит серию Jupyter Notebooks для разведочного анализа данных (EDA), а также задач регрессии и классификации

## Описание проекта

В работе используются данные о химических соединениях. Для соединений представлены числовые химические дескрипторы и целевые показатели:
- `IC50, mM` - показатель противовирусной активности;
- `CC50, mM` - показатель цитотоксичности;
- `SI` - индекс селективности, рассчитываемый как отношение `CC50 / IC50`.

В рамках работы решаются следующие задачи:

- регрессия для `IC50`;
- регрессия для `CC50`;
- регрессия для `SI`;
- классификация: `IC50` выше медианы выборки;
- классификация: `CC50` выше медианы выборки;
- классификация: `SI` выше медианы выборки;
- классификация: `SI > 8`.
